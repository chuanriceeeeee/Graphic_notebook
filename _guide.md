1. 建立工业管线视野（快速扫盲）
想要快速了解大厂现在的渲染管线到底在用什么技术，B站是最好的金矿。

腾讯游戏学院“TA百人计划” (B站)

怎么用： 不要全看（内容太多），直接挑“图形渲染”相关的初期课程。比如讲 PBR 基础、渲染管线发展史的几节课。

为什么推荐： 这是国内大厂 TA 体系最系统的一次知识开源。它能帮你把脑海里的零散概念（比如你写过的深度测试、光照计算）跟现代商业引擎的模块名称对上号。

2. 攻克 NPR/二次元渲染（米哈游对口核心）
既然目标是米哈游，二次元卡通渲染（NPR）是你绕不开的护城河。你需要看业内的“圣经”。

GDC 演讲：《Guilty Gear Xrd's Art Style : The Gear is in the Details》

怎么用： B站有熟肉中文字幕版。拿个本子记下他们是如何处理面部阴影（修改顶点法线）、如何做描边（外扩法）、如何控制内部阴影边界的。

为什么推荐： 这是 3D 动漫渲染的开山之作，无数国内二次元游戏的渲染思路都源自于此。看懂了这个，面试时聊起 NPR 你就能引经据典。

搜索关键词：“贺甲 渲染”

怎么用： 找他之前在各大技术沙龙的图文分享或演讲录像。

为什么推荐： 他曾是米哈游的渲染技术核心，他的技术分享能让你最直观地感受到一线二次元大厂正在解决什么级别的图形学痛点。

3. Shader 源码速通实战（代码落地）
不要拿大部头的书一页页翻，直接上机看别人写好的高质量、结构清晰的代码。

Catlike Coding (英文原站或知乎中文翻译版)

怎么用： 这是一个基于 Unity 的神级教程。直接看它的 "Rendering" 或 "Custom SRP" 系列。新建一个工程，跟着他的代码一行行敲。

为什么推荐： 作者的代码极其优美且富有逻辑。对于习惯严谨代码结构的 C++ 开发者来说，看着他在 C# 和 HLSL 之间游刃有余地传递数据，你会迅速顿悟 Shader 的编写套路。

知乎专栏：浅墨的图形学（《Real-Time Rendering》提炼）

怎么用： 当你在实战中遇到不懂的 PBR 概念或数学公式时，把它当字典查。

为什么推荐： 纯理论的 RTR4 太厚了，浅墨大神的专栏把最核心的数学公式和工业界做法提炼成了人话，非常适合查漏补缺。

4. 实习面经突击（对齐考试大纲）
一个月的时间，你的学习方向必须由“考点”来驱动。

牛客网 / 游戏魂 / 游资网

怎么用： 疯狂搜索“米哈游 TA 面经”、“腾讯 客户端 渲染实习 面经”。把近两年的高频问题复制下来，整理成一个 Excel 表格。

为什么推荐： 面试官常问的问题其实很集中（比如：前向渲染与延迟渲染的优缺点、抗锯齿算法 MSAA vs TAA 的原理、半透明物体排序问题）。把这些高频考点搞懂，你的知识网络会自动交织起来。
